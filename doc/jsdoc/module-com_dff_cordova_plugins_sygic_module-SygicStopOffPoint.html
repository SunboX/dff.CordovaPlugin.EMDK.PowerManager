Class: SygicStopOffPoint

com/dff/cordova/plugins/sygic/module~ SygicStopOffPoint

new SygicStopOffPoint(isSearchAddress, isVisited, pointType, location, offset, id, iso, caption, address)

Represent a stop of an itinerary.
Parameters:
Name Type Description
isSearchAddress boolean Flag determining whether address should be searched (for example when adding waypoint).
isVisited boolean Flag determinig whether waypoint was already visited.
pointType Number Type of stop Sygic.model.SygicStopOffPoint.pointType
location SygicPosition SygicPosition of the stop.
offset Number Road offset from our map, where the waypoint is located. Use conversion table to obtain the offset from map provider segment ID. When using offset, the ISO code has to be used.
id Number Unique id of stop.
iso string Iso code of country the stop is located.
caption string Name of stop.
address string Address of stop.
